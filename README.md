# heRomod

A fork of the [HEEMOD](https://github.com/pierucci/heemod) package for the development of Markov cohort and partitioned survival models by [Jordan Amdahl](https://github.com/jrdnmdhl) and [PAI](http://www.pai2.com).  HEEMOD was oringally developed by [Kevin Zarca](http://www.urc-eco.fr/Kevin-ZARCA,402) and [Antoine Filipović-Pierucci](https://pierucci.org).
